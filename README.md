# Infoundr Site 

## Getting started:

1. Clone the repo: 
```bash
git clone https://github.com/infoundr/infoundr-site.git
```

2. Move to the frontend directory: 
```bash
cd src/frontend
```

3. Install dependencies: 
```bash
npm install
```

4. Run the development server: 
```bash
npm run start
```
